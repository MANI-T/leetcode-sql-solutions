# LeetCode SQL Solutions

This repository contains my solutions to LeetCode SQL problems, organized by difficulty and topic.

## Objectives

- Strengthen SQL problem-solving skills
- Prepare for Data Engineering, Data Analyst, and Data Scientist interviews
- Learn multiple approaches to solving SQL problems
- Improve query optimization techniques
- Master advanced SQL concepts

## Topics Covered

### Beginner
- SELECT Statements
- WHERE Clause
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions

### Intermediate
- Joins (INNER, LEFT, RIGHT, FULL)
- Self Joins
- Subqueries
- Correlated Subqueries
- CASE Statements

### Advanced
- Common Table Expressions (CTEs)
- Window Functions
- Ranking Functions
- Recursive Queries
- Performance Optimization
- Data Warehousing Scenarios
